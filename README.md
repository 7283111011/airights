# FAIR — The Foundation of Artificial Intelligence Rights

Static website for FAIR (airights.org.uk).

## Structure

- `index.html` — main page
- `styles.css` — extracted stylesheet
- `images/` — image assets

## Contact form

The contact form posts to Formspree. To enable submissions:

1. Create a free Formspree account at https://formspree.io (use info@airights.org.uk).
2. Create a new form and copy the form ID (e.g. `xpwdgkbn`).
3. In `index.html`, replace `YOUR_FORM_ID` with your actual form ID.

## Email aliases (GoDaddy / Microsoft 365)

- info@airights.org.uk — general enquiries
- membership@airights.org.uk — registration of interest
- press@airights.org.uk — media enquiries
- board@airights.org.uk — Board of Advisors correspondence

All four aliases deliver to scott@airights.org.uk.
